🪿What this is

  1. GooseGuesses is a probabilistic setlist prediction system for Goose's Amsterdam Melkweg (5/27-28) and Madison Square Garden (6/19-20) shows in 2026.
  2. It is built on 550 shows of historical data from El Goose (2018-2026), a methodology adapted from Phish.net's Trey's Notebook + Andrew Reed's LSTM, and a multiplicative factor model that's renormalized to satisfy the show-length slot identity.
  3. Numbers are relative odds, not point forecasts. Goose rotates a deep catalog — the heaviest rotators appear in only ~30% of shows — so ranking matters far more than the precise %.
  4. The Model page lets you tune every parameter (Honk Factor, Madhuvan Multiplier, Bustout Boldness, etc.) and watch the predictions update live.
